How It Works

        

          This dashboard is protected by @warpy-auth-sdk/core running on Fastify. The authentication flow:
        

        
    
          
  1. User clicks "Sign in with Google" on the login page
    2. 
          
  2. Fastify route /auth/signin/google calls authenticate() from the SDK
    3. 
          
  3. SDK redirects to Google OAuth consent screen
    4. 
          
  4. Google redirects back to /auth/callback/google
    5. 
          
  5. SDK exchanges the code for tokens and creates a JWT session
    6. 
          
  6. Session cookie is set and user is redirected to dashboard
    7. 
          
  7. Dashboard fetches session via /auth/session endpoint
    8.
